Pseudo-code gameplay :

idées : 
	- bouton pour activer/désactiver les aides ?

-------------------------------------------
Fonction appelée quand l'utilisateur clique
-------------------------------------------
	
Si clic dans plateau:
	Si pas de pion sélectionné:
		Si clic sur pion qui peut bouger :
			sélectionner pion
			erreurs=0
			Si priseObligatoire() not null:		// on a pas encore sélectionné de pion, on teste du coup si il y a prise obligatoire
				mouvements possibles = priseObligatoire()
				(bool) prise obligatoire = true
			Sinon:
				mouvements possibles = canMove()

		Sinon:		// on a cliqué comme de la m*rde
			erreurs ++
			
	Sinon (si pion sélectionné):
		Si clic dans mouvements possibles:		// on déplace le pion
			déplacer pion
			
			Si (bool) prise obligatoire:
				manger pion adverse
				Si PriseObligatoire() not null:		// il faut continuer la prise
					sélectionné = nouvelle position
					mouvements possibles = priseObligatoire()
				Sinon:
					(bool) prise obligatoire = false
					déselectionner le pion
					actualiseTour(pion joué)
			Sinon:
				déselectionner le pion
				actualiseTour

		Sinon:
			erreurs++

aide
----
Si sélectionné is not null:
	Si prise obligatoire:
		halo rouge sur pion(s) à prendre
	Sinon:
		halo bleu sur les cases dispos
		
	halo jaune sur pion(s) à jouer
Sinon :
	Si prise obligatoire:
		halo jaune sur les pions à jouer
	Sinon: 
		halo vert sur les pions joua
